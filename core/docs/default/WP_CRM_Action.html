Class WP_CRM_Action

Description
Description | Vars (details) | Methods (details)

Actions are linked to Events. Every time an event is fired, it's attached actions are checked and run.

This class is linked to the /actions folder whick contains action templates. Need to run ::scan() static method to update the action list.
