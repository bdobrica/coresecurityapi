Class WP_CRM_App

Description
Description | Vars (details) | Methods (details)

Class handling home-screen apps.

Each app is defined in the /themes/wp-crm/static/{slug}.php file. The structure of the {slug}.php file is:

  1.  <?php
  2.  &#47;*
  3.  App Title: string
  4.  App Description: string
  5.  App Size: int
  6.  App Style: string
  7.  *&#47;
  8.  ...
  9.  ?>
